# Booking Request Workflow

## Goal
Capture a booking request from a website and send it into a simple workflow for review and follow-up.

## Workflow example
Website booking form → webhook → Make → email or spreadsheet

## Main steps
1. A user fills in a booking or request form
2. The form sends the information to a webhook
3. Make receives the data
4. The request is stored or forwarded
5. A follow-up action is triggered

## Use case
Useful for service businesses, consultations, booking requests and small business websites that need a simple inquiry flow.
